# 🌸 SidBloom - Flower Boutique

A beautiful, responsive flower shop website built with React. Browse, search, filter, and sort through 45+ beautiful flowers!

![SidBloom Screenshot](https://via.placeholder.com/800x400/9A1750/FFFFFF?text=SidBloom)

## ✨ Features

- 🌸 **45+ Flowers** - Browse a wide collection of flowers
- 🔍 **Search** - Find flowers by name in real-time
- 🏷️ **Filter** - Filter flowers by category (Roses, Tulips, Sunflowers, etc.)
- 💰 **Sort** - Sort by price (low-high, high-low) or name
- 📱 **Responsive** - Works on desktop, tablet, and mobile
- 🎨 **Beautiful UI** - Custom color palette with smooth animations
- ⚡ **Optimized** - Uses React hooks (useState, useMemo) for performance
