# 🌸 SidBloom - Flower Boutique

A beautiful, responsive flower shop website built with React. Browse, search, filter, and sort through 45+ beautiful flowers with a stunning custom color palette.


## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌸 **40+ Flowers** | Browse a wide collection of flowers across 9 categories |
| 🔍 **Search** | Find flowers by name in real-time |
| 🏷️ **Filter** | Filter by category (Roses, Tulips, Sunflowers, Lilies, Daisies, Lavender, Orchids, Peonies, Mixed) |
| 💰 **Sort** | Sort by price (Low-High, High-Low) or name (A-Z) |
| 📱 **Responsive** | Perfect on desktop, tablet, and mobile |
| 🎨 **Custom Colors** | Beautiful color palette with smooth animations |
| ⚡ **Optimized** | Uses React hooks (useState, useMemo) for performance |
| 💖 **Interactive** | Hover effects, animations, and smooth transitions |
